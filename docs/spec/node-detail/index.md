# node-detail

**ID:** node-detail
**Spec:** Settled
**Implementation:** Complete

The node detail page (`/nodes/:name`): conditions, a consumed-vs-free resource usage indicator (cpu, memory, pods), addresses, labels, the pods scheduled on the node (with sortable CPU/Memory bar columns and View-mode/Value-format toggles showing each pod's share of the node), node events, and a Performance tab with CPU/Memory utilization cards above the per-node treemap.

## Sub-features
None.
