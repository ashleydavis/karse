# quick-find

**ID:** quick-find
**Spec:** Settled
**Implementation:** Partial

Command-palette-style quick-pickers in the header. Two are shipped: a context quick-picker (`Ctrl+K`) and a namespace quick-picker (`Ctrl+Shift+K`), each a searchable dropdown. The roadmap's wider "quick-find across all resource kinds" (a dropdown that searches every resource and navigates to its detail page) is **not** yet shipped, so this feature is Partial.

## Sub-features
None.
