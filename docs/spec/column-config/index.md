# column-config

**ID:** column-config
**Spec:** Settled
**Implementation:** Complete

Per-table, persisted configuration of which columns are visible and the order they show in. A "Columns" button on every resource table opens a modal with Visible and Hidden sections; columns drag to reorder within Visible and drag between sections to show or hide. The choice is saved per table in `localStorage` and survives navigation and reload.

## Sub-features
None.
