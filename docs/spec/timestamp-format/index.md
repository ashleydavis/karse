# timestamp-format

**ID:** timestamp-format
**Spec:** Settled
**Implementation:** Complete

A header toggle that switches every timestamp Karse displays between its age (the relative time since it happened, e.g. `3h`) and the absolute local time (e.g. `14 Jul 2026, 09:23:45`). The choice is app-wide and persisted.

## Sub-features
None.
